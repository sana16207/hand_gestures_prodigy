# Hand Gesture Recognition

A machine learning based hand gesture recognition system using MediaPipe and Scikit-learn.

## Features
- Detects hand landmarks using MediaPipe
- Classifies gestures using ML
- Supports multiple gestures:
  - A
  - B
  - C
  - D
  - fist
  - okay
  - palm
  - thumbsup

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Scikit-learn
- NumPy
- Matplotlib

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## Output
Gesture prediction window with classified hand signs.
